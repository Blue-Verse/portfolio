# USER_STORY_codex

## 주요 사용자

1. 한국 기업/기관 의사결정자
2. 해외 프로젝트 가능성을 보는 파트너
3. Blue Verse의 작업 방식을 알고 싶은 잠재 고객

## 사용자 스토리

- 의사결정자는 첫 화면에서 Blue Verse가 신뢰형 제품을 다뤄 본 팀인지 빠르게 판단하고 싶다.
- 방문자는 전체 `market-intel` 프로젝트 범위를 한 페이지에서 훑고, 관심 있는 프로젝트만 상세하게 열어 보고 싶다.
- 방문자는 별도 페이지에서 팀 소개, 작업 방식, 프로젝트 범위를 차분하게 확인하고 싶다.
- 사용자는 과장된 마케팅 문구 대신 시장, 기간, 주요 성과, 운영 난이도 같은 실질 정보를 보고 싶다.
- 사용자는 작은 로고, 비어 보이는 카드, 느슨한 레이아웃 때문에 사이트가 미완성처럼 보이지 않기를 원한다.
